# Usage 

```
git clone https://github.com/Sunjung-Dev/fastAPI.git
```

```
pip3 install -r requirements.txt
```

```
build
docker build --tag fastapitest:0.1 .
```

```
run
docker run --name fastapi_test -d -p 8000:8000 fastapitest:0.1
```
